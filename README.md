# KeyOwn Habitat | The Future of Home Ownership

A premium Home Ownership Advisory System (HOAS) designed to transition tenants into homeowners through a rent-to-own financial framework. 

This repository contains the landing page and interactive front-end for the KeyOwn Habitat platform.

## Features
- **Modern UI/UX**: Premium aesthetic with dark mode and vibrant accent colors.
- **Interactive Calculator**: Real-time Rent vs. Asset Wealth comparison calculator.
- **Financial Framework**: Detailed interactive timeline of the HOAS homeownership journey.
- **Mobile Responsive**: Fully optimized, swipeable sliders, and dynamic layouts for all devices.
- **Dynamic Animations**: Canvas-based ambient backgrounds and smooth scroll reveals.

## Technologies Used
- HTML5
- CSS3 (Vanilla, custom properties, animations)
- JavaScript (Vanilla, IntersectionObserver, Canvas API)
- Phosphor Icons

## Running Locally
To preview the website locally, run a simple HTTP server in the project directory:
```bash
python3 -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.
